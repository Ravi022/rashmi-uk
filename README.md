# Rashmi Metaliks UK Ltd. — Premium React Website

World-class corporate frontend built with React + Vite + Tailwind + Framer Motion + React Router + Lucide Icons.

## Tech Stack
- React (Vite)
- Tailwind CSS
- Framer Motion
- React Router
- Lucide Icons

## Project Structure

src/
- components/
- sections/
- pages/
- data/
- assets/
- hooks/
- utils/

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Notes
- Content is preserved from the provided HTML prototype and reorganized into reusable React architecture.
- All pages include dynamic SEO title/description/OpenGraph tags through a reusable hook.
- Fully responsive across mobile, tablet, and desktop.
